# 昆蟲世界 · Insect World

寫實昆蟲教學模擬器 — 從台灣到世界各大洲,44 種昆蟲即時活動。你可以**切換 9 大區域**、**遠看**整片生態,也能點選任何一種昆蟲**近看**牠的身體構造、真實數據與生命週期。

純前端、繁體中文、零安裝。所有昆蟲與貼圖都是載入時用 Three.js 幾何體 + Canvas 程序化生成,整個專案**沒有任何圖片檔**。

## 特色

- **兩層視角**:生態全景(自由環顧、縮放)↔ 昆蟲特寫(相機飛近)。
- **構造標註**:近看時在 3D 模型上標出頭、胸、腹、複眼、翅、足、口器等部位並解說。
- **教學資料卡**:每種昆蟲的真實體長/翅展/壽命、生命週期(完全/不完全變態)、趣聞。
- **生命週期 3D 演示**:一鍵播放卵→幼蟲→蛹→成蟲(或卵→若蟲→成蟲)的立體形態演變 —— 毛蟲、雞母蟲、水蠆、懸蛹、繭都做成 3D,相機隨階段自動取景。
- **昆蟲導覽**:自動巡覽當前區域的昆蟲,依分類故事順序解說。
- **真實相對比例**:瓢蟲(7mm)與長戟大兜蟲(170mm)以真實體型比並存,建立尺寸感。

## 9 大區域 · 44 種昆蟲

**台灣**(12 種):大紫蛺蝶、獨角仙、台灣扁鍬形蟲、西方蜜蜂、黑棘蟻、無霸勾蜓、台灣熊蟬、七星瓢蟲、黑翅螢、寬腹螳螂、台灣大蝗、台灣皮竹節蟲。

**世界**(各 4 種):日本・東亞、東南亞(蘭花螳螂、皇蛾…)、南亞・中東(月亮蛾、沙漠飛蝗…)、非洲(歌利亞大角金龜…)、歐洲(深山鍬形蟲、帝王偉蜓…)、北美洲(帝王斑蝶、十七年蟬…)、中南美洲(藍閃蝶、長戟大兜蟲、子彈蟻…)、大洋洲(巨沙螽、紐西蘭發光蟲…)。

橫跨鱗翅目、鞘翅目、膜翅目、蜻蛉目、半翅目、螳螂目、直翅目、竹節蟲目,涵蓋完全/不完全變態、擬態、生物發光、社會性、長程遷徙等主題。

## 執行

```
node tools/serve.mjs 8137
# 開 http://localhost:8137/
```

## 操作

- 滑鼠拖曳環顧、滾輪縮放。
- 點左側圖鑑或畫面中的昆蟲 → 近距離觀察。
- 底部可切換「構造標註」「動作」,或啟動「昆蟲導覽」。
